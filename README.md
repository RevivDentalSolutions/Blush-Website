# Blush Ink & Beauty Studio Website

Static Next.js marketing site for a luxury permanent makeup studio.

## Update quick links
- Booking URL and contact info: `data/site.ts`
- Service names/pricing: `data/site.ts`
- Gallery placeholders: `app/gallery/page.tsx`

## Run locally
```bash
npm install
npm run dev
```

## Build static export
```bash
npm run build
```
Output is generated for static hosting (Vercel/Cloudflare Pages compatible).
